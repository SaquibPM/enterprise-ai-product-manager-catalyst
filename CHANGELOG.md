## [1.0.0] - 2026-04-20

### Added — Complete Marketplace Launch

All 8 plugins fully built with production-grade content:

- **37 skills** with embedded frameworks, worked examples, and enterprise-specific guidance (28,000+ lines)
- **13 commands** with skill chain documentation, data handoff specs, and parallelization guidance
- **13 sample outputs** demonstrating real enterprise deliverables (PRDs, competitive briefs, sprint plans, GTM plans, strategy briefs, agent design documents)
- **6 cross-plugin workflows** with full step-by-step content
- **111 eval test cases** across all skills (3 per skill)

### Plugin Content

- **pm-agentic-ai** — 5 skills (6,011 lines): agentic-architecture, ai-evaluation, guardrails-design, human-in-loop, ai-native-ux. Includes worked examples for invoice processing, spend classification, contract analysis, PO approval routing, and spend analytics anomaly detection.
- **pm-definition** — 5 skills (4,130 lines): prd-development, api-spec-design, prioritization-frameworks, roadmap-planning, compliance-review. Includes RICE scoring with 5-feature worked example, complete PRD template, API versioning strategy.
- **pm-operations** — 5 skills (4,356 lines): stakeholder-comms, decision-records, vendor-evaluation, platform-strategy, incident-response. Includes complete post-mortem template, vendor scorecard with 3-vendor evaluation, ADR for RAG vs fine-tuning.
- **pm-discovery** — 5 skills (3,379 lines): customer-interviews, competitive-analysis, research-synthesis, market-sizing, cab-management. Includes Teresa Torres interview framework, TAM/SAM/SOM analysis, 3-competitor S2P competitive brief.
- **pm-launch** — 5 skills (3,365 lines): gtm-planning, sales-enablement, release-notes, rfp-response, customer-migration. Includes launch tier framework, complete battlecard, migration runbook for 200 customers.
- **pm-delivery** — 4 skills (3,125 lines): sprint-planning, acceptance-criteria, analytics-instrumentation, tech-debt-negotiation. Includes capacity planning formulas, 12-criterion acceptance criteria example, tech debt ROI calculation.
- **pm-leadership** — 4 skills (2,128 lines): vision-strategy, okr-management, executive-presentations, team-coaching. Includes PM competency framework, complete Q2 OKR set, QBR deck outline.
- **pm-growth** — 4 skills (1,632 lines): metrics-analysis, experimentation, pricing-packaging, expansion-strategy. Includes metric tree framework, B2B experimentation with small-N guidance, land-and-expand playbook.

### Workflows

- feature-kickoff — Discovery → Spec → Design → Sprint (with parallelization)
- quarterly-planning — OKR → Prioritization → Roadmap → Comms (audience-specific outputs)
- launch-sequence — GTM → Enablement → Rollout → Metrics (T-8 weeks through T+4 weeks)
- competitive-response — Urgent competitive threat response with severity-based escalation
- ai-feature-lifecycle — Complete AI feature lifecycle with mandatory eval and compliance gates
- customer-discovery — 4-6 week discovery cycle with loop-back capability
